# 项目概述

这是一个名为"重生之我是福州古厝修复者"的网页游戏项目。玩家扮演古建筑修复者，依次修复福州的三处历史文化街区：上下杭、朱紫坊、三坊七巷。每个章节包含战斗和修复两个阶段。

## 技术栈

- 后端：Django (Python)
- 前端：原生 HTML/CSS/JavaScript（单页应用）
- 数据存储：Session（无数据库）

## API 接口(localhost:10203)

### GET /api/progress/
获取当前游戏进度。

返回示例：
```json
{
    "chapter": 0,
    "chapter_name": "上下杭",
    "finished": false,
    "fight_won": false,
    "repaired": false,
    "relics": {
        "shangxiahang": false,
        "zhuzifang": false,
        "sanfangqixiang": false
    }
}
```

### POST /api/progress/advance/
推进游戏进度。

请求体：
```json
{ "action": "fight_win" | "repair_complete" | "reset" }
```

返回更新后的进度状态（格式同上）。

## 游戏流程

1. 每章节：战斗胜利 → 修复完成 → 获得魂器 → 进入下一章
2. 章节顺序：上下杭(0) → 朱紫坊(1) → 三坊七巷(2) → 通关(3)
3. 必须先 fight_win 才能 repair_complete
4. 任意时刻可 reset 重置
