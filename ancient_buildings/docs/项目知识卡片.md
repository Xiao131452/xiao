# 古建筑项目 · 知识卡片（精简版）

与当前实现一致的四点摘要：

1. **数据文件**  
   使用最新版 `src/data/building_coords.json` 替换旧版即可；冗余测试脚本与过时说明已清理，主应用只依赖规范化后的单一数据源。

2. **图片交互**  
   - **首页轮播**：5 张，数据取前 5 条建筑；箭头、指示点、自动播放、**左右滑动**。  
   - **详情侧栏**：每条建筑 **最多 5 张**图；箭头、指示点、**左右滑动**。

3. **信息字段**  
   在原有形制、朝代等基础上，界面明确展示 **「建筑类型」**（`building_type`）与 **「百度地址」**（`baidu_address_url`，可由数据字段或地址/坐标生成）；百科为 **「百度百科词条」**（`baike_url`）。

4. **链接**  
   百科、百度地图类链接与 JSON 及 `normalizeBuilding.js` 生成规则一致，均为 `https://` 外链，新窗口打开。

详细字段表与目录结构见同目录上一级 **`MAP_MODULE_README.md`**。
