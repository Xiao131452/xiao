# 古建筑地理位置可视化系统

一个专业的古建筑地理位置数据可视化仪表板，用于展示和分析中国古建筑的分布情况。

## 功能特性

### 界面设计

- 深蓝色主题 + 玻璃效果（Glassmorphism，可在地图 Tab 样式中逐步对齐）
- 响应式布局设计
- 平滑的动画过渡效果

### 筛选功能

- **朝代筛选**：按朝代过滤古建筑
- **省份筛选**：按省份过滤古建筑
- **保存状态筛选**（Vue 主站）：保存完好 / 重建
- **实时结果统计**：显示当前筛选条件下的建筑数量、活跃筛选标签

### 地图交互

- **古建筑点位**：散点/热力，颜色与保存状态等维度关联（`BuildingHeatmap.vue` + ECharts Geo）
- **全国模式省份聚合**：未筛选时以省份为单位的标记与下钻
- **悬停与点击**：悬停简介卡片；点击建筑进入侧栏详情；点击省份进入省级视图与列表
- **独立演示版**（若使用纯 HTML）：可用红点 + 省份旗标 + 点击弹出卡片；与主站 ECharts 地图为两种实现路径，数据字段应对齐 `building_coords.json` 与 `normalizeBuilding.js`

### 统计图表

- **统计分析 Tab**：总数、省份数、朝代数、形制数卡片 + 朝代/类型条形分布
- **仪表板式右栏**（与独立 HTML 对齐时）：总体环形/仪表盘、朝代折线、省份 TOP5 柱状图，数据源建议与 `filteredBuildings` 一致

## 数据说明

- 主数据：`src/data/building_coords.json`
- 规范化：`src/utils/normalizeBuilding.js`（`images`、`building_type`、百科与百度地图链接、`details.overview` 等）
- 地图组件接收的 `buildings` 应为规范化后的列表
